# monsters-rolodex

## Installation

Clone the project from GitHub and install dependencies

`https://github.com/mohamedibrahim1998/monsters-rolodex.git`
`cd monsters-rolodex`
`npm install`

Then run the project with
`npm start` or `yarn start`
